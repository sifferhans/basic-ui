# Heading

Use for defining sections on a document.

```vue
<script setup lang="ts">
import { Heading } from '@sigveh/basic-ui'
</script>

<template>
  <Heading :level="3">This is a h3 tag</Heading>
  <Heading :level="4">This is a h4 tag</Heading>
</template>
```

```ts

}
```
